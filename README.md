# Inicio

## Ejecución del programa


1. Asegúrate de tener un entorno virtual de Python activo. Si no tienes uno, puedes crearlo y activarlo con:

**En Unix/macOS:**
```sh
python3 -m venv venv
source ./venv/bin/activate
```

**En Windows (cmd):**
```bat
python -m venv venv
venv\Scripts\activate.bat
```

```
2. Instala las dependencias necesarias:
pip install -r requirements.txt
```


3. Ejecuta el programa usando make:

```sh
make run
```
